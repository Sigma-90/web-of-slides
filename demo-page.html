1Branched Slides

You can have more than one link per slide and each one can define which slide is shown next. That way you can create slideshows that are able to branch out, depending on the viewers choice.

Next ›

2Autoplay

You can choose to play all slides automatically. The time a slide is shown before the next one is brought up can be defined for each slide individually.

Next ›

2.1Pitfalls of Autoplay

The autoplay feature will save the user a few clicks, but might irritate people in case the next slide is shown before they finished reading or if they have to wait too long for the next slide. It also linearizes the animation, as it can't branch out anymore. Unless you pause it, like here.
